How to set up in Gmail

  1. Click Open next to your name above.
  2. Select everything on the page (Cmd+A on Mac, Ctrl+A on PC).
  3. Copy it (Cmd+C / Ctrl+C).
  4. In Gmail, go to Settings (gear icon) → See all settings.
  5. Scroll to Signature and click Create new.
  6. Paste into the editor (Cmd+V / Ctrl+V).
  7. Click Save Changes at the bottom.
